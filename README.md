# GnomeWin TopBar

A minimalist and functional top bar for Windows, inspired by GNOME aesthetics.

Features

Smooth Interface: Native animations for all panel elements.

Calendar Widget: Convenient calendar access by clicking on the date.

System Integration: Displays the current keyboard layout, battery level, and network activity/status.

Live Equalizer: The panel automatically detects music playback and displays a real-time visualizer.

Customization: Flexible settings for transparency and custom color schemes.

Technologies

The application is written in C# using WPF (Windows Presentation Foundation). It utilizes system hooks (Win32 API) to interact with the OS, ensuring the panel operates stably and smoothly without interfering with your workflow.

Installation

Download the latest version from the Releases section.

Run the .exe file.

The application will automatically dock to the top of your desktop.

Usage & Configuration

Left-Click on Date: Opens the calendar widget.

Left-Click on System Tray: Opens the Quick Settings / Control Center.

Right-Click on the Panel: Opens the Settings menu to change color and transparency.

Adding Programs: Click the "+" button in the app dock to pin your favorite .exe files or shortcuts.

Removing Programs: Hover over a pinned app and click the red "−" button to remove it from the dock.
